OpenJDK

OpenJDK是Java平台（Java SE）的开源实现。

  • OpenJDK主页

  • OpenJDK github存储库

RISC-V状态

两个独立的团队开始在OpenJDK端口上工作。

BishengJDK团队由Wang Yadong领导。 他们已经发布了第一个用于RV64G的HotSpot JIT端口。

PLCT团队由史宁宁领导。

双方都在共同的 RISC-V端口 上进行合作并开展工作。

可以在 此处 找到OpenJDK / HotSport RV64G JIT移植的状态 他们对实现其100倍加速 目标 非常乐观

PHP

PHP是一种针对Web开发的编程语言。

  • PHP主页

RISC-V状态

PHP解释器可在RISC-V上使用。 但是，某些功能（例如PCRE JIT）尚未移植，并且目前在代码库中没有特定于RISC-V的优化。 因此，目前无法实现最佳性能。

Rust

Rust是一种为性能，可靠性和生产力而设计的编程语言。它依靠LLVM生成代码。

  • Rust主页

  • Rust手册

RISC-V状态

自1.42.0（2020-03-12）起，Rust对RISC-V RV64（riscv64gc-unknown-linux-gnu）提供了第2层支持。支持RV32，但是currenlty没有完整的标准库。

  • https://doc.rust-lang.org/nightly/rustc/platform-support.html

嵌入式Rust

Rust Embedded是一个用于在“裸机”嵌入式系统上启用Rust编程语言的项目。

  • Rust嵌入式工作组

  • 嵌入式Rust Book

RISC-V状态

由于Embedded Rust不需要Rust的标准库（no_std），因此与完整的Rust运行时相比，可用的RISC-V平台更多。

支持以下平台：

  • 裸机RISC-V RV32

    • RV32I

    • RV32IMAC

    • RV32IMC

  • 裸RISC-V RV64

    • RV64IMAFDC

    • RV64IMAC

还有一个快速入门指南，可为HiFive1开发板构建Rust应用程序：

  • https://github.com/riscv-rust/riscv-rust-quickstart
