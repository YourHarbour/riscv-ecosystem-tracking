什么是RISC-V


RISC-V是一种免费的开放式ISA，通过开放标准的协作实现了处理器创新的新时代。

RISC-V ISA在架构上提供了新的免费，可扩展的软件和硬件自由度，为未来50年的计算设计和创新铺平了道路。

RISC-V International是总部位于瑞士的全球非营利协会。

RISC-V成立于2015年，最初是RISC-V基金会，现有29名成员，现已成为一个真正的全球性组织，在50多个国家/地区拥有1k +成员。

RISC-V打破了半导体行业的壁垒，召集了不同的公司，行业和地理位置进行开放式协作。

RISC-V将模块化技术方法与开放许可业务模型结合在一起，这意味着任何地方的任何人都可以利用RISC-V International贡献和生产的IP。

RISC-V International在瑞士成立，是一家真正的全球性组织。作为非营利组织，RISC-V International对产品或服务没有任何商业利益。

RISC-V也不代表任何地理地区的政治立场。世界各地的组织在这个处理器创新的新时代中携手合作。

RISC-V的应用领域

可用于大型计算机、家用计算机、智能手机和微小型嵌入式系统等，可以广泛地应用于各种处理器的设计中。
